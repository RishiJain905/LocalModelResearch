# Contamination-checks

## 📌 Overview

Methods for detecting benchmark data contamination (benchmark leakage) in LLM training data. Covers matching-based, black-box, and watermarking-based detection approaches.

> *"Exposure of a large language model to benchmark data during the training process leads to distorted evaluation results."*
> — [Benchmark Data Contamination Survey](https://arxiv.org/html/2406.04244v1)
