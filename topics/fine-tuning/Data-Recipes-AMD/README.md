# Data-Recipes-AMD

## 📌 Overview

Synthetic data generation pipelines, datasets, and tools for fine-tuning LLMs on AMD GPUs (ROCm). Covers production-grade ecosystems like SAND, LuminaSFT, and Unsloth-based synthetic data workflows.

> *"Synthetic data generation on AMD GPUs enables creating high-quality training datasets without human-labeled data, using the ROCm stack on MI300X/MI250 accelerators."*
