# Unsloth-Optimization

## 📌 Overview

This section covers Unsloth AI and the custom kernel optimization techniques that power it. Unsloth is an open-source LLM fine-tuning framework that achieves 2-5x speedup and 30-90% VRAM reduction through custom Triton kernels, LoRA/QLoRA, and smart padding-free packing.

> **Note**: The correct organization name is **Unsloth AI** (commonly just "Unsloth"), not "Unsloth Studios."
