## 🔑 Key Themes

### Bolt.new
- **WebContainers** — Node.js running entirely in browser via WebAssembly/SharedArrayBuffer
- $40M ARR, 1M monthly active users, 15 engineers
- Claude 3.5 Sonnet was the critical breakthrough enabling production-ready code generation
- v2: 98% fewer error loops, 1000x bigger project capacity, multi-agent

### Emergent
- **5-agent pipeline**: Architect → Designer → Developer → Integration → PM (QA)
- Kubernetes ephemeral pods — each session = one full Linux environment with root
- $100M raised, $300M valuation, targeting $100M+ ARR
- Full-stack: React + FastAPI + MongoDB, portable code (no vendor lock-in)

### Vibe-Coding Templates
- Cursor Rules + MCP servers = reusable vibe-coding workflows
- "Four Pillars": Context, Atomic Tasks, Memory, Patterns
- PRD → Tech Design → Build pipeline templates
- Best practices: plan mode, git branches per feature, code review before merge
